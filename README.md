Dubai Guide Bot



Dubai Guide Bot is an AI-powered trip planning assistant that helps users create a personalized travel plan for visiting Dubai. The bot interacts with users by asking questions about their preferences, interests, and budget, then generates a suitable travel plan with activity and place suggestions. Built using Python, Streamlit, and the OpenAI API, it provides an easy-to-use chatbot interface where users can get instant recommendations and trip ideas.

How to Use



* Clone the repo.
* Install the dependencies from requirements.txt.
* Add your own API key in the .env file or inside app.py (where the key is used).
* Run the app using: streamlit run app.py





Start asking anything about Dubai.
